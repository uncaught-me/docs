Overlay - это версия приложения, которое требует установки на ваш ПК. На данный момент приложение доступно только для Windows и macOS.

## Возможности приложения

- Чат с несколькими нейросетями в один момент (gpt-4o, gpt-4o-mini, claude-3-5-sonnet, claude-3-7-sonnet, gemini-2.5-pro)
- Прослушивание и транскрибация голоса собеседущего с последующей генерацией ответа при помощи нейросетей.
- Полная скрытность при показе экрана в звонках.

## Установка для Windows

Скачать инсталлер можно [здесь](https://uncaught.me/overlay/guide).

## Установка для macOS

Скачать инсталлер можно [здесь](https://uncaught.me/overlay/guide). Для macOS потребуется установка дополнительных компонентов. Подробная инструкция представлена ниже.

### Установка Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Выполните команду в терминале.

### Добавление Homebrew в PATH

```bash
echo "export PATH=/opt/homebrew/bin:$PATH" >> ~/.bash_profile && source ~/.bash_profile
```

Выполните команду в терминале. Если вы используете ZSH, то вместо ~/.bash_profile используйте ~/.zshrc.

### Установка SoX (утилита, повзоляющая записывать звук с разных источников)

```bash
brew install sox
```

### Установка BlackHole (аудиодрайвер, который позволит записывать системный звук)

```bash
brew install blackhole-2ch
```

После установки BlackHole, вам может потребоваться перезагрузить компьютер и настроить аудио в системных настройках.

### Настройка Audio-MIDI

После установки BlackHole, необходимо создать многовыходное аудиоустройство для записи системного звука:

1. Откройте приложение "Настройка Audio-MIDI" (находится в папке Utilities или можно найти через Spotlight)
2. В меню "Окно" выберите "Показать аудиоустройства"
3. Нажмите на "+" в левом нижнем углу и выберите "Создать многовыходное устройство"
4. В появившемся списке выберите ваше основное аудиоустройство и BlackHole 2ch
5. Для основного аудиоустройства поставьте галочку в столбце "Master Device"
6. Переименуйте устройство (например, в "System and BlackHole")

После создания многовыходного устройства, выберите его в качестве устройства вывода звука в системных настройках. Теперь весь системный звук будет направляться как в ваши колонки/наушники, так и в BlackHole, что позволит записывать системный звук.

### Снятие карантина с приложения

После установки приложения, необходимо снять ограничения macOS с помощью команды:

```bash
sudo xattr -cr "/Applications/uncaught.app"
```

Эта команда позволит запустить приложение без предупреждений системы безопасности macOS.
